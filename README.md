# ASP.NET Travel Site

A full-featured travel booking web application built with ASP.NET Core MVC.

## Features
- Search and book hotels, flights, rental cars, and events
- Step-by-step vacation package creation
- RESTful Web API integration for hotel data and reservations
- SQL Server database with stored procedures
- Secure payment handling with AES encryption
- CAPTCHA-based security for login and registration
- Session-based user navigation

## Tech Stack
- ASP.NET Core MVC
- Razor Views
- C#
- SQL Server
- RESTful Web APIs
- HTML, CSS, JavaScript
